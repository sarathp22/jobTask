student registration instructions EJS tempate used.

1.git clone

2.take cmd on current directory

3.npm install

4.open mongodb server installed in the local machine.

5.npm start will load index page.(http://localhost:3000/)

6.http://localhost:3000/register 
Api to register teacher.

7.http://localhost:3000/login 
Api to login teacher.

8.http://localhost:3000/student
Api to enter students marks.

9.http://localhost:3000/studentModify
views table contents button to edit and delete data entered.

10.http://localhost:3000/studentModify/:id
modify a perticular student entry

11.http://localhost:3000/studentEdit/:id
delete a perticular student entry

12.http://localhost:3000/result
a public api to view student marks or results.

Note: ejs not supported put and delete verbs so post and get only used to create apis.
Json token generated in the console.
